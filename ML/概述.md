
# 模型评估与选择
**模型**
假设空间：$F ={f|Y=f_\theta(X),\theta \in  \mathbb{R}^n}$ Y: 输出空间变量, X: 输入空间变量 $\rightarrow$ 所有可能的条件概率分布或决策函数的集合, 如 $f = \omega^Tx$
方法 = 模型 + 策略 + 算法
**策略**
- 如何从假设空间中选择合适的策略 $\rightarrow$ loss function or cost function
- $L(Y,f(X))$ usually > 0 
- loss function
	- 0-1 Lf  
	- quadractic : $\vert Y-f(x)\vert_2$
